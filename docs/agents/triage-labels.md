# Triage Labels

The engineering skills use five canonical triage roles.
This table maps those roles to the GitHub labels used by this repository.

| Canonical role    | GitHub label      | Meaning                                       |
| ----------------- | ----------------- | --------------------------------------------- |
| `needs-triage`    | `needs-triage`    | A maintainer needs to evaluate the issue      |
| `needs-info`      | `needs-info`      | The issue is waiting for reporter information |
| `ready-for-agent` | `ready-for-agent` | The issue is fully specified and agent-ready  |
| `ready-for-human` | `ready-for-human` | The issue requires human implementation       |
| `wontfix`         | `wontfix`         | The issue will not be actioned                |

When a skill refers to a canonical triage role, use the corresponding GitHub label from this table.
